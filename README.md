# Target Brazil E-Commerce Analysis Using SQL

## Project Overview

This project analyzes Target's Brazil e-commerce operations using SQL. The dataset contains over 100,000 orders placed between 2016 and 2018.

## Business Problem

Target wants to understand customer behavior, order trends, payment methods and delivery performance to improve business operations.

## Tools and Technologies

* SQL
* MySQL

## Analysis Performed

* Order Trend Analysis
* Monthly Seasonality Analysis
* Customer Distribution by State
* Revenue Analysis
* Freight Cost Analysis
* Delivery Performance Analysis
* Payment Type Analysis
* Installment Analysis

## Key Insights

* Order volume increased over time.
* Customer distribution varies significantly across states.
* Freight costs differ among regions.
* Payment preferences influence purchasing behavior.

## Business Recommendations

* Improve logistics in high delivery time regions.
* Optimize freight costs.
* Focus on states with higher customer concentration.
* Enhance customer experience through efficient delivery.

## SQL Concepts Used

* Joins
* Aggregate Functions
* Group By
* Date Functions
* Common Table Expressions (CTEs)
* Case Statements
